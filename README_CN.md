# LinCore 简介

&emsp;&emsp;一个基于vkguide教程的Vulkan代码实验平台，基于Vulkan1.3的新特性，将会实现教程中的所有功能，并且将会基于《Mastering Graphics Programming with Vulkan》进一步拓展该引擎，并实现更多图形学算法。

# 已实现特性
- [x] 自动管线布局创建
- [x] 简易CVar系统
- [x] 无绑定资源管理
- [x] 管线缓存
- [x] 多线程记录命令
- [x] 简易SceneGraph
- [x] Microfacet BRDF (Cook-Torrance)
- [x] GPU驱动渲染
- [x] 延迟渲染

# 待实现特性
- [ ] 异步计算
- [ ] IBL
- [ ] 阴影映射
