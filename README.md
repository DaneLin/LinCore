# LinCore 简介
&emsp;&emsp;一个基于vkguide教程的Vulkan代码实验平台，基于Vulkan1.3的新特性，将会实现教程中的所有功能，并且将会基于《Mastering Graphics Programming
with Vulkan》进一步拓展该引擎。
